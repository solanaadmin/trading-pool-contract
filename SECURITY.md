# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the SolanaFund Trading Pool smart contract, please report it responsibly.

**Contact:** admin@solanafund.io

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and work to resolve confirmed issues promptly.
